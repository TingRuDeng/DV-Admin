## 简介
基于RBAC模型权限控制的中小型应用的基础开发平台，前后端分离架构
前端采用Vue3 + Vite + JavaScript + Element-Plus，后端Django + Django-rest-framework。

JWT认证，支持swagger

-----

## 前置准备
```angular2html
Python >= 3.11.0（如多版本环境推荐使用pyenv进行管理）
nodejs >= 20
Mysql >= 8.0 (可选，默认数据库sqlite3，支持5.7+，推荐8.0版本)
Redis (可选，最新版)
```
